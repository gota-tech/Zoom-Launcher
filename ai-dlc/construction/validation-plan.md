# Validation Plan

## Functional Checks

- 会議を登録できる
- 登録した会議が一覧に表示される
- 参加ボタンでURLが開く
- 会議を編集できる
- 会議を削除できる
- ページ再読み込み後も保存データが残る

## UI Checks

- デスクトップ幅で表示が崩れない
- モバイル幅で表示が崩れない
- 長い会議名でもレイアウトが壊れない
- パスコードが意図せず丸見えにならない

## Security Checks

- 実データをリポジトリに保存していない
- サンプルデータはダミー値だけである
- 保存方式の注意点がREADMEに書かれている
