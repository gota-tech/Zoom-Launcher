# Runbook

## Local Use

1. `docs/index.html` をブラウザで開く
   - macOSでは `open-zoom-launcher.command` をダブルクリックしてもよい
2. 会議情報を登録する
3. 参加ボタンからZoom会議を開く
4. パスコードが必要な場合は `PW表示` または `PWコピー` を使う
5. 登録内容を変更する場合は `編集` を押してフォームで更新する
6. テストデータや不要なデータを消す場合は `全削除` を使う

## Backup

1. `書き出し` を押す
2. ダウンロードされたJSONファイルを安全な場所に保存する
3. JSONファイルにはZoom URL、ミーティングID、パスコードが含まれるため、共有しない
4. 書き出しJSONはGitや公開フォルダに置かない

## Recovery

1. `読み込み` を押す
2. バックアップ済みJSONファイルを選ぶ
3. 置き換え確認でOKを押す
4. 会議一覧が復元されたことを確認する

## Data Location

- 会議情報はブラウザのLocalStorageに保存される
- ブラウザ、プロファイル、端末が変わると保存内容は共有されない
- ブラウザデータを削除すると保存内容も消える可能性がある

## Cleanup

1. 必要なら `書き出し` でバックアップする
2. `全削除` を押す
3. 確認ダイアログでOKを押す
4. 一覧が空になったことを確認する
